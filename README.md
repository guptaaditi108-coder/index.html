# OCD Detection - Basic HTML App

A **simple educational web app** for basic OCD screening.  
⚠️ **Disclaimer:** This is **not a clinical diagnosis tool**.

## Features
- 5-question self-screening
- Simple scoring (0–20) with basic interpretation
- Color-coded results
- Fully implemented in HTML, CSS, JS
- Lightweight, runs in any browser

## Usage
- Open `index.html` locally in browser
- Or deploy using **GitHub Pages** (Settings → Pages → main branch → root folder)
- URL: https://<your-username>.github.io/ocd-detector-basic/

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
